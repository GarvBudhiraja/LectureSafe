# LectureSafe 🎓

**Live Demo:** [garvbudhiraja.github.io/LectureSafe](https://garvbudhiraja.github.io/LectureSafe/)

Hi! I'm Garv, and I'm currently a student. This is my very first web development project! 

I built **LectureSafe** to solve a problem that most of us students face regularly: keeping track of attendance without having to constantly log in to slow official college portals. I wanted a fast, straightforward calculator that not only shows my current percentage but also tells me exactly how many upcoming classes I can safely miss, or how many I need to attend to recover.

## Why I Built This

For my first project, I wanted to build something that solves a real, daily problem while keeping privacy in mind. There is no backend, no database, and no data collection. Everything runs locally and safely right inside your browser.

## Features

- **Instant Calculation:** Enter your numbers and instantly see your current percentage.
- **Safe Misses:** Find out exactly how many upcoming lectures you can skip without dropping below your target attendance.
- **Recovery Planner:** If your attendance is low, the app calculates the exact number of consecutive classes you need to attend to get back in the safe zone.
- **100% Private:** No login, no cookies, no tracking. Your numbers are yours alone.
- **Clean UI:** Fully responsive design with Dark and Light mode support.

## How It's Built

Since I am focusing on building a strong technical foundation, I created this project using purely core web technologies without relying on any heavy frameworks:
- HTML5
- CSS3 (Custom properties, flexbox, grid, and animations)
- Vanilla JavaScript

## For Developers (Running Locally)

If you want to check out my code or run the project on your own machine, it's very easy since it's a static site:

1. Clone this repository:
```bash
   git clone [https://github.com/garvbudhiraja/LectureSafe.git](https://github.com/garvbudhiraja/LectureSafe.git)
   ```
2. Open the `index.html` file in any web browser. That's it—no server setup or package installations required!
